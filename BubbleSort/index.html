    冒泡排序

    算法思想

    冒泡排序属于典型的交换排序, 也就是通过交换符合条件的元素来进行排序. 在冒泡排序中, 交换的是相邻元素.

    假设有数组:

    位置 0 1 2
    值 $ a $ $ b $ $ c $

    以升序为例, 通过比较相邻元素的大小, 先比较0, 1位置若

    a > b, 则交换a, b的值
a <= b, 则不交换

    $ a, b $比较之后, 前进一位到位置1, 接下来比较位置1, 2. 一直进行比较和交换直到数组尾. 这就是冒泡的一趟排序, 这个过程其实可以看成把最大值移动到数组尾的过程, 由于一趟排序不能保证已经升序, 所以之后还需要继续.

    到什么时候结束呢?

    由于每一趟都是把当前最大值放到每趟最后, 所以每一趟都可以少比较一次, 一个比较粗暴的方法就是比较(假设数组有$ n $个元素) $ n $-1趟. 当然也可以优化一下, 其实只要在一趟排序中没有一次交换, 也就是已经排好序了.

    算法性能

    最好 最坏 平均 空间 稳定
    $ O $($ n $) $ O $($ n^2 $) $ O $($ n^2 $) 1 稳定

    算法实现

    Python3 实现

    def bubbleSort(arr: list):
    length = len(arr)
    for i in range(length-1):
        flag = True
        for j in range(length-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                flag = False
        if flag:
            break
    return arr
