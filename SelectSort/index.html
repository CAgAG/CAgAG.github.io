    选择排序

    算法思路

    从头至尾扫描序列，将当前位置到序列尾看成一个区间, 在区间内找出最小的一个元素，与区间的第一个元素交换，接着从剩下的元素中继续这种选择和交换方式，最终得到一个有序序列。举个例子:

    位置 0 1 2 3
    值 2 1 3 4
    区间 $ i $

    $ i $到序列最后组成一个区间, 在位置0-3中找出区间的最小值, 也就是值1(位置1), 找到后将位置1与区间第一个位置(位置0)进行交换, 然后$ i $向后移一位, 指向位置1组成新的区间. 一直重复, 直到$ i $到序列尾.

    算法性能

    平均时间复杂度 最好情况 最坏情况 空间复杂度 稳定
    $ O(n^2) $ $ O(n^2) $ $ O(n^2) $ $ O(1) $ 不稳定

    算法实现

    Python3 实现: 

    def selectSort(arr: list):
    length = len(arr)
    for i in range(length):
        minIndex = i
        for j in range(i+1, length):
            if arr[minIndex] > arr[j]:
                minIndex = j

        arr[minIndex], arr[i] = arr[i], arr[minIndex]
    return arr

    链式选择排序实现

    在链表中实现选择排序需要用到2个指针, 要注意指针的结束条件, 本次以交换值来实现
