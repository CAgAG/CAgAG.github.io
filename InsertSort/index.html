    插入排序

    算法思路

    通过构建有序序列，将一个未排序的数据，插入到已排序序列中(保持有序).

    例如(升序):

    位置 0 1 2 3 4
    值 0 9 10 6 5
    区间 $ i $

    起始位置到$ i $的区间为已构建好的有序序列, 从未排序区间(从$ i+1 $到序列尾), 选择一个数, 一般是$ i+1 $位置的数(值6), 将数插入到已构建好的有序序列, 也就是位置0和1之间, 此时就需要将位置1, 2都向后移一位, 再将位置3的值覆盖位置1的值, $ i $ 再向后移一位. 一直重复上述操作直到$ i $到序列尾.

    算法性能

    平均时间复杂度 最好 最坏 空间复杂度 稳定性
    $ O(n^2) $ $ O(n) $ $ O(n^2) $ $ O(1) $ 稳定

    算法实现

    Python3实现: 

    def InsertSort(arr: list):
    length = len(arr)
    for i in range(1, length):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

    return arr

    链式插入排序实现

    在单链表中实现时注意指针的使用, 本次实现用到三个指针, cur表示上述表中的$ i+1 $, pre指向cur待插入的位置, next指向cur的下一个位置, next是为了记录cur, 用于下一趟循环.
